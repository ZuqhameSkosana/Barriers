BarrierTest Instructions and Makefile

   Make sure you are in the folder BarrierS

1. type command: make clean
2. compile the classes by typing : make
3. To run the BarrierTest class run the command: make run1 
  
  please note there are 5 tests in the makefile you can choose from run1 to run5, they have different values
  for example first test which is run one is run1: java BarrierTest 5 5, run2 is : java BarrierTest 10 10  etc.
  
4. else to run the BarrierTest class run the command: java BarrierTest 5 5

5. to make java docs run the command: make doc
